# DATABASE SETUP GUIDE

Sau khi mở venv thì chạy: python manage.py setup_db --reset

Đợi setup, không đụng tay vào terminal, nếu vô tình hủy setup thì chạy lại code, tiếp tục làm việc sau khi thấy 2 dòng này:

✅ Database seeded successfully
✅ Database setup complete!

Nếu không thấy cả 2 thì chụp ảnh báo lại.
